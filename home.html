Health Monitoring & Tracking System

  • Designed in three integrated modules for real-time health tracking.
  • Module 1: Hardware setup with sensors to monitor body parameters, using Raspberry Pi 3 as the processor.
  • Module 2: Backend development using Node.js and MongoDB for data storage and server integration with the website.
  • Module 3: Combines hardware and backend to display live health status and location on the website, ensuring smooth data flow from sensors to database to UI.
Individual Role:
  • I was primarily responsible for coding and development tasks.
  • This included programming the Raspberry Pi to interface with various sensors, and designing the database system that receives input from those sensors and pushes data to the live dashboard.

STOCK PRICE PREDICTION

  • Objective: Develop a predictive model to forecast stock prices for major companies (Apple, Boeing, Microsoft, Intel, IBM) using historical data analysis.
  • Methods:
    • Linear Regression: A simple model that predicts outcomes by fitting a straight line to minimize the error between predicted and actual values.
    • Decision Tree: A model that predicts outcomes by splitting data into branches based on feature values, creating a tree-like structure.
    • Random Forest: An ensemble model that combines multiple decision trees to improve accuracy and reduce overfitting by averaging their predictions.
  • Motivations:
    • Understand stock market dynamics and patterns.
    • Enhance predictive accuracy for investment decisions.
    • Analyze the impact of market variables on stock performance.
    • Advance financial analysis with innovative forecasting methods.
